# Movie Series
Sample SolidJS-PWA movie series app.

# Init
## Frontend
```sh
pushd frontend
# 'pn' stands for 'pnpm'
pn install
popd
```

## Backend
```sh
pn install
```

# Running Locally
## Frontend
```sh
pushd frontend
pn run dev
popd
```
## Backend
```sh
pn run dev
```

# Release
```sh
./front-build
```

# Deploying to Heroku
```sh
./deploy 'new commit message'
```
